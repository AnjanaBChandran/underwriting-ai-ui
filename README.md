# Underwriting AI UI (Lovable Build)

This repository contains the frontend interface for the AI underwriting assistant, built using Lovable.

---

## 🚀 What This Is

- UI layer for interacting with the underwriting AI system  
- Built using vibe coding via Lovable  
- Designed for underwriters to review documents and generate summaries  

---

## 🧠 Context

This UI connects to a larger AI system that performs:

- OCR (document extraction)  
- Document classification  
- Rule-based checks  
- LLM-based summarization
